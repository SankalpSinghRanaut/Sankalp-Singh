### 🏭 Inventory & Demand-Supply Performance Analysis

<img align="right" src="https://media.giphy.com/media/26tn33aiTi1jkl6H6/giphy.gif" width="280" alt="Analytics GIF"/>

> **July 2025** &nbsp;|&nbsp; [🔗 View on GitHub](https://github.com/SankalpSinghRanaut/Inventory-and-Demand-Supply-Performance-Insights)

![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white)
![Power BI](https://img.shields.io/badge/Power%20BI-F2C811?style=flat-square&logo=powerbi&logoColor=black)
![Pandas](https://img.shields.io/badge/Pandas-150458?style=flat-square&logo=pandas&logoColor=white)
![Excel](https://img.shields.io/badge/Excel-217346?style=flat-square&logo=microsoftexcel&logoColor=white)

| 🏷️ | Outcome |
|:---:|:--------|
| 💰 | **Audited USD 2.3M+ in revenue across 5,001 orders** using end-to-end SQL EDA — profiling margin distributions, outlier discounts, and loss drivers leadership lacked visibility into |
| 🔴 | **Identified 21.1% of orders generated negative profit margins** — Tables category alone responsible for **−USD 17,700 in losses** driven by a 42.8% discount rate silently eroding the bottom line |
| 📈 | **Modelled dynamic pricing restructuring** in Python — projecting **USD 4,000–6,000 in annual loss recovery** by capping unprofitable discounting thresholds |
| 🔄 | **Surfaced a critical retention gap:** 80%+ one-time buyers vs. top 5 customers contributing USD 50K+; proposed CLV-based loyalty segmentation |
